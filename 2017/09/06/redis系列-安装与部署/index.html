redis系列--linux下安装与部署

  • 2017-09-06

导语

Redis 是当前比较热门的NOSQL之一

1. redis的安装

  • download    https://redis.io/download
  • scp local_file    scp -rp redis-4.0.1.tar.gz root@ip:/tmp
  • tar file    tar -zxvf /tmp/redis-4.0.1.tar.gz -C /home/
  • complie && install   
       cd /home/redis-4.0.1/  make
       cd src/  make install
      

2. redis的部署

  • 文件管理    方便配置文件和cmd管理，创建bin和conf文件:
    在安装目录下：mkdir bin | mkdir ect
                            mv src/mkreleasehdr.sh redis-check-aof redis-check-rdb redis-cli redis-sentinel redis-server redis-benchmark ./bin/
                            mv redis.conf /ect
  • 启动服务(前台)    ./bin/redis-server
                     注意：默认情况下，启动的redis服务实在前台直接运行的，当关闭会话后，服务也随之关闭。
  • 启动服务(后台)    修改conf文件      vim /ect/redis.conf    将daemonize的值改为yes.
                                ./bin/redis-server ect/redis.conf     |   
    停止服务 bin/redis-cli shutdown (or pkill bin/redis-server)
  • 启动客户端    ./bin/redis-cli
  • 查看端口号    netstat -tunpl | grep 6379

3. redis 测试

  • 启动客户端    ./bin/redis-cli
                  > set testkey “this is my test”
                  > get testkey
                  > “this is my test”
                  > exit

    至此，Redis的安装部署全部成功，开始享受Redis之旅了.
